# 💻 01 — Coding

Programming practice and reference notes. Coding is one of the three foundations of
cybersecurity (alongside networking and system administration) — it's what lets you
read exploits, write tools and scripts, automate tasks, and think through problems
the way attackers and defenders do.

## What this folder covers

- **Data structure references** — how each core structure works, its complexity, and
  copy-paste-ready code.
- **LeetCode problem write-ups** — solutions grouped by pattern (Arrays & Hashing,
  Two Pointers, …), each explained step by step with traces and diagrams. Problem from 150 neetcode

## Structure

| Path | What's inside |
|------|---------------|
| [`Python/DataStructure.md`](Python/DataStructure.md) | Reference for core data structures (arrays, linked lists, stacks, queues, hash maps/sets, …) with code and complexity. |
| [`Python/ArrayAndHashes.md`](Python/ArrayAndHashes.md) | LeetCode write-ups — Arrays & Hashing pattern. |
| [`Python/2pointer.md`](Python/2pointer.md) | LeetCode write-ups — Two Pointers pattern. |

> 📌 Each write-up follows the same format: **Problem → Idea → Approach → Complexity
> → Solution → How it works** (with traces/diagrams where useful).
