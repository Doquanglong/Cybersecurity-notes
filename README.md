# Cybersecurity-notes 
A personal knowledge base where I document what I learn across cybersecurity — coding, penetration testing, security operations, and networking. These are working notes: written as I study, practice, and break things in labs, then refined over time.

Purpose

This repo exists to:


Keep my learning organized and searchable in one place
Turn scattered notes, commands, and writeups into a structured reference
Track my progress as I move deeper into each area
Serve as a quick cheat sheet I can come back to during labs and CTFs
